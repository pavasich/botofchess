# !giveaway

`!giveaway` initiates a raffle that requires participants to say the
command `!enter` in chat.
Bot then picks a winner from all participants at random.

---
#### Usage
`!giveaway`__*`<thing>`*__

|Parameter|Type|
|---|---|
|__thing__|_text_|

__Examples__:

`!giveaway gamecode`

`!giveaway cookies`

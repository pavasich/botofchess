# !roll

Roll some dice

---

#### Usage
`!roll`__*`<number_of_sides>`*__

`!roll`__*`<number_of_rolls>`*__`d`__*`<number_of_sides>`*__

|Parameter|Type|
|---|---|
|__number_of_sides__|_number_|
|__number_of_rolls__|_number_|

__Examples__:

`!roll 20`
> roll a 20-sided die

`!roll 2d10`
> roll 2 10-sided dice and sum the results
